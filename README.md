# Description

This is a [Nest](https://github.com/nestjs/nest) project to build a simple API that allows users to create, read, update, and delete locations.

# Getting started


## Installation

```bash
$ npm install
```

## Running the app

Copy `.env.example` to `.env`

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Visit [`localhost:3000/api`](http:localhost:3000/api) to check out the API docs
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
